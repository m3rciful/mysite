Для запуска программы необходимо подготовить базу данных.
В phpmyadmin создайне нового пользователя с именем "mysite", пароль ему задайте "mysite".
Отметьте  чекбокс: "Создать базу данных с таким же именем и предоставить на неё все привилегии.".
Зайдите в созданную базу и выберите вкладку "Импорт", укажите файл в папке сайта "mysite.sql".
Добавится таблица для работы базового функционала.
Аналогично, добавьте таблицы из файла "kurator.sql" и "user.sql".
Всё, база готова к работе с программой.
